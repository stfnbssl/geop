    History DBs

    The History DBs are JSON databases of historical people, events and concepts plus an HTML+Javascript application for their visualization.

    Both the App and the JSON DBs are generated by the Wizzi Factory from text document edited with common code editors (at present i am using VSCODE). The text documents must be conformant to their Wizzi Schema.

    The History DB is part of the GEOP geopolitical+software project hosted on the GITHUB platform . It is an open source project with a MIT license and collaborations are welcome.
